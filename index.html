My Projects

Travel Planning App

Developed a dynamic travel planning app that enables users to input a location, travel dates, and hotel name to retrieve weather forecasts, location coordinates, relevant images, and location data (capital, population). The system integrates with multiple APIs (Geonames, Weatherbit, Pixabay, Restcountries) to provide real-time data, with features like local storage, trip sorting by start date, expired trip highlighting, and the ability to print or export trips (individually or in bulk) to PDF. Built using Express server and Webpack.

Movie Site

An online movie guide that displays the newest movies, some details about them, and a link to watch them. I used The Movie Database (TMDB) API for the data, and the website was built using React with Redux technology.
